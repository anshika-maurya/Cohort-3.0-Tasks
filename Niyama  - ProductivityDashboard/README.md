# 🌿 Niyama

### A clean and modern productivity dashboard built with HTML, CSS & JavaScript.

Plan your day • Manage tasks • Stay focused • Track your goals

## ✨ Features

### 📊 Dashboard
- Live Date & Time
- Weather Widget
- Motivational Quote
- Dynamic Dashboard Cards
- Dark / Light Mode

### ✅ Tasks
- Add, Complete & Delete Tasks
- Mark Important Tasks
- Live Dashboard Updates
- Local Storage Support

### 📅 Planner
- Add, Edit & Delete Events
- Planner Notes
- Hourly Timeline
- Dashboard Synchronization

### ⏱ Focus Timer
- Pomodoro Timer
- Focus / Short Break / Long Break
- Progress Ring
- Timer State Persistence

### 🎯 Daily Goals
- Add & Complete Goals
- Live Progress Tracking
- Dashboard Updates

### ⚙️ Settings
- Change Display Name
- Upload Profile Picture
- Reset All Data
- Theme Preference Saved
