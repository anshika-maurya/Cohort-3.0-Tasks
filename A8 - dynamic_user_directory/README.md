# Dynamic User Directory

A dynamic user directory built using **HTML**, **CSS**, and **JavaScript**. The application displays user cards from JSON data and allows users to be added, edited, and deleted through an interactive form.

## 🚀 Features

* Display users from JSON data
* Add new users through a form
* Edit existing user information
* Delete users from the directory
* Dynamic DOM rendering
* Event Delegation for efficient event handling
* Responsive card-based UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* DOM Manipulation

## ⚙️ How It Works

1. User data is stored in an array of objects.
2. The `renderUsers()` function dynamically creates user cards.
3. New users can be added using the form.
4. Existing users can be edited using the Edit button.
5. Users can be removed using the Delete button.
6. The UI updates automatically whenever the data changes.
