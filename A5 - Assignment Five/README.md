# Dribbble UI Clone

A modern Dribbble-inspired UI built using HTML and CSS.

## Features

* Responsive design
* Dribbble-style animation effects
* Clean and modern UI
* Flexbox-based layout
* Smooth visual composition and spacing

## Concepts Practiced

* Flex-box
* Responsive Design
* CSS Animations
* Visual Hierarchy
* Layout Structuring

## Tech Stack

* HTML5
* CSS3

## Live Demo

[View Project](https://cohort-3-0-tasks-zc2u.vercel.app/)

## Purpose

Built this project to improve frontend development skills by recreating a modern UI design, adding interactive animations, and making the layout responsive across different screen sizes.
