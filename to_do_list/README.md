# 📝 Todo List App

A simple and interactive Todo List application built using HTML, CSS, and Vanilla JavaScript (DOM Manipulation).

## 🚀 Live Demo

🔗 Live Link: [Open Todo List](https://cohort-3-0-tasks-z5lg.vercel.app/)

## 📌 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Clean and responsive UI
* Dynamic DOM manipulation
* Instant task management without page refresh

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (DOM)
