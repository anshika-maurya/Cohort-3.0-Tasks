# 🚀 NextTask

A modern and responsive task management dashboard built with **HTML, CSS, and JavaScript**.

Track projects, manage tasks, monitor progress, and stay productive through a clean and interactive user experience.

### ✨ Features

* Project & Task Management
* Task Status Tracking
* Search & Filtering
* Dynamic Dashboard Statistics
* Dark Mode Support
* Responsive Design
* Modal-Based Forms
* Vanilla JavaScript CRUD Operations

### 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
