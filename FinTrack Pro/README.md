# 💰 FinTrack Pro

A modern and responsive personal finance tracker built with **HTML, CSS, and JavaScript**. Track your income and expenses, visualize cash flow, and manage your finances with an intuitive dashboard.

## 🌐 Live Demo

🔗 **Live:** https://cohort-3-0-tasks-rhcx.vercel.app/

---

## ✨ Features

- 🔐 Login & Register (Local Storage)
- 💳 Add Transactions
- ✏️ Edit Transactions
- 🗑️ Delete Transactions
- 📊 Dynamic Dashboard Summary
- 📈 Cash Flow Analysis (Chart.js)
- 👤 Profile Settings
- 💱 Currency Preference
- 🌙 Dark / Light Mode
- 🔄 Reset All Transaction Data
- 💾 Persistent Data using Local Storage
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Local Storage API
- Remix Icons

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/FinTrack-Pro.git
```

Go to the project folder

```bash
cd FinTrack-Pro
```

Open `index.html` in your browser.
