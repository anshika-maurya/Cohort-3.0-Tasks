# Two Leaves and a Bud - Responsive Website Clone

A responsive clone of the "Two Leaves and a Bud" website built using HTML and CSS.

## Overview

This project was created to practice responsive web design and improve frontend development skills by recreating a modern website layout.

## Features

* Responsive design for different screen sizes
* Clean and modern user interface
* Flexbox-based layouts
* Mobile-friendly structure
* Smooth visual hierarchy and spacing

## Technologies Used

* HTML5
* CSS3

## Concepts Practiced

* Responsive Web Design
* Flexbox
* Layout Structuring
* UI Development
* Spacing and Alignment
